# Prompt for LLM

This file is the prompt content for the LLM.  
Before making any operations, the LLM must first review the README.md file to understand the entire project.  
Before making any operations, the LLM must first load LLM.md as basic instructions to avoid making partial modifications to the project.  
After completing operations, ensure that the Chinese and English versions of configuration files have corresponding content and consistent order.  
After completing operations, ensure that the scripts in both language versions have consistent logic and accurate translation.  
After completing operations, ensure that the documentation files in both language versions have accurate content and consistent translation.  

# LLM 提示词

本文件是给 LLM 使用的提示词内容。  
在做任何操作前，LLM 需要先查看 README.md 文件，理解整个工程。  
在做任何操作前，LLM 需要先载入 LLM.md 作为基础的指令，避免对工程做局部修改。  
做完操作之后，注意检查中英文版本的配置文件是否内容对应，顺序一致。  
做完操作之后，注意检查中英文版本的脚本代码是否逻辑一致，翻译准确。  
做完操作之后，注意检查中英文版本的说明文件是否内容准确，翻译一致。
